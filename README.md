# ERA - REgame Awake

Un atelier pédagogique interactif pour sensibiliser les jeunes aux dark patterns dans les jeux vidéo.

---

## Description

ERA (REgame Awake) est un dispositif hybride qui combine :
-  Un jeu de plateau physique
-  Une interface web interactive

L'objectif : aider les collégiens (10-14 ans) à comprendre et déjouer les techniques de manipulation financière (dark patterns) utilisées dans l'univers du jeu vidéo.

---

### Durée
- 45 minutes par atelier
- Jusqu'à 5 participants par partie
